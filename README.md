# Minesweeper
A simple Minesweeper game that is written by C++ Using [CLUI](https://github.com/SBU-CE/clui) library
## Installation
1. Clone the directory
2. Install Mingw
3. put all these files on folder.
4. Enjoy the game
## Features
* Save game
* Play with 'W' 'A' 'S' 'D'
* Play with givinig coordinates
* pick & put up flag on game
